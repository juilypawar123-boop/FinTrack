# Personal Finance Tracker

A modern, interactive **Personal Finance Tracker Web Application** built with **HTML, CSS, and JavaScript**. Track income and expenses, visualize financial data with charts, and maintain a detailed transaction history. Designed to showcase a professional, real-world web project for portfolio purposes.

---

## **Live Demo** ##

Watch live demo on Netlify : https://fintrack201.netlify.app

-----

## **Features**
 
1.	Full-stack CRUD
2.	Responsive UI
3.	Basic Charts / Analytics
4.	Data persistence
5.	Validation & Error Handling

- Add new income and expense transactions.
- Interactive pie charts showing income vs. expenses.
- Dashboard with total income, total expenses, and balance.
- Transaction history table with filters:
  - By category
  - By type (Income / Expense)
  - By month
  - Search by keyword
- Export transaction history to CSV.
- Fully responsive and mobile-friendly design.
- Uses **localStorage** for persistent client-side storage.

---

## **Technologies Used**
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Chart.js** for dynamic visualizations
- **LocalStorage** for client-side data persistence
